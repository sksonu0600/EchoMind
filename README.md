# EchoMind

> 🚧 **Status:** Work in Progress

EchoMind is an AI-powered desktop assistant that understands voice, context, and user intent to automate desktop workflows.

## Features

- 🎤 Voice transcription
- 🤖 LLM integration
- 🖥️ Desktop automation
- 🧠 AI memory (Coming Soon)
- 🌐 Browser automation (Coming Soon)
