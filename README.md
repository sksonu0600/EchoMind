# EchoMind

> 🚧 **Status:** Work in Progress

EchoMind is an AI-powered desktop assistant that leverages voice interaction, AI agents, and desktop automation to streamline everyday workflows.

## ✨ Features

- 🎤 Voice transcription
- 🤖 LLM-powered conversations
- 🖥️ Desktop automation
- 🧠 Persistent AI memory *(Coming Soon)*
- 🌐 Browser automation *(Coming Soon)*
- 🔗 MCP integration *(Coming Soon)*

## 🛠️ Tech Stack

- Python
- FastAPI
- Electron
- LangGraph
- OpenAI
- Whisper

## 🚀 Getting Started

```bash
git clone https://github.com/sksonu0600/echomind.git
cd echomind
```

### Backend

```bash
cd backend
pip install -r requirements.txt
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
